# チーム 11 レシピアプリ

## 技術スタック

### フロントエンド

- Next.js(App Router)
- Tailwind / shadcn/ui

### バックエンド

- supabase
- Prisma

## npm script

```
# 開発モード起動
npm run dev
# ビルド
npm run build
# 本番モード起動
npm run start
# コードチェック
npm run lint
# コード整形
npm run format
# コードを修正
npm run lint:fix
```
